## Quick start

<Steps>
  <Step title="Choose a platform library">
    Pick the official Flic library for your target platform — iOS, Android, Windows, Linux, or embedded C. See [Platform Libraries](/platform-libraries) for links and setup instructions.
  </Step>
  <Step title="Scan and pair a button">
    Use the library's scan wizard to discover nearby Flic buttons in Public mode and complete the cryptographic pairing process. See [Pairing and Connecting](/guides/pairing-and-connecting).
  </Step>
  <Step title="Subscribe to button events">
    After pairing, send an `InitButtonEventsLightRequest` packet to begin receiving click, hold, and double-click notifications. See [Button Events](/guides/button-events).
  </Step>
  <Step title="Handle events in your app">
    Parse incoming `ButtonEventNotification` packets using one of the four trigger mode use cases (up/down, click/hold, single/double click, or all three). See [Button Event Packets](/protocol/button-event-packets).
  </Step>
</Steps>
